# Publix Coupon Clipper

![coupon](coupon.png)

## How To Install

### Repo

 ```pip install git+<https://github.com/Chrisae9/PublixCoupons.git#egg=coupon[extra]>```

### Terminal

 ```pip install .```

### GUI

 ```pip install .[gui]```

### Dev

 ```pip install -e .[gui,dev]```

## Note

If you're using zsh you need to escape square brackets instead type:

 ```pip install -e .\[gui\]```
